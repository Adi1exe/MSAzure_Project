About This Project

A modern web application leveraging Microsoft Azure's powerful Cognitive Services for seamless speech-to-text and text-to-speech conversion.

Key Features

Real-time Speech Recognition

Convert spoken words into text instantly with Azure's advanced speech recognition technology.

Natural Voice Synthesis

Transform text into natural-sounding speech with multiple voice options including male and female voices.

Modern UI/UX Design

Beautiful, intuitive interface with dark mode support and smooth animations for enhanced user experience.

Interactive Visualizations

Dynamic visual feedback with animated voice visualizer that responds to recording status.
